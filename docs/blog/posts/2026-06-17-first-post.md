# 첫 번째 글: 블로그를 시작하며

MkDocs와 Material 테마로 블로그를 시작했습니다. 이 글은 블로그가 잘 동작하는지
확인하기 위한 예시 글입니다.

<!-- more -->

## 글 쓰는 법

새 글은 `docs/blog/posts/` 폴더에 마크다운 파일로 추가하면 됩니다.
파일 맨 위 `---` 사이에 아래 정보를 적어주세요.

```yaml
---
date: 2026-06-17      # 작성일 (필수)
categories:
  - 일반              # 카테고리
tags:
  - 태그예시          # 태그
---
```

`<!-- more -->` 위쪽 내용이 목록에서 미리보기로 보입니다.

## 코드 블록 예시

```python
def hello(name: str) -> str:
    return f"안녕하세요, {name}님!"

print(hello("준수"))
```
