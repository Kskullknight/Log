# 마크다운 사용법

자주 쓰는 마크다운 문법 정리입니다.

## 제목

```markdown
# 제목 1
## 제목 2
### 제목 3
```

## 강조

```markdown
**굵게**, *기울임*, ~~취소선~~
```

**굵게**, *기울임*, ~~취소선~~

## 목록

```markdown
- 항목 1
- 항목 2
  - 하위 항목

1. 첫째
2. 둘째
```

## 링크와 이미지

```markdown
[링크 텍스트](https://example.com)
![이미지 설명](image.png)
```

## 표

```markdown
| 이름 | 값 |
| ---- | -- |
| A    | 1  |
| B    | 2  |
```

| 이름 | 값 |
| ---- | -- |
| A    | 1  |
| B    | 2  |

## 인용

```markdown
> 인용문입니다.
```

> 인용문입니다.

## 강조 박스 (Material 전용)

```markdown
!!! note "제목"
    내용을 적습니다.
```

!!! note "제목"
    내용을 적습니다.
